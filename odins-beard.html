Odin's Beard

Platform Game

Game Creation Society Project

Project Lead: Joey Perrino

github code

September - December, 2018

Description

Odin's Beard is a platform game with two levels. The objective of the game is to kill the enemies. There are flying enemies and also enemies on the ground. When a player is attacked by the enemy, the player loses a life. However, the player can gain back a life by powering up with a heart.




My Role

For this project, my task was to program the enemies' behavior. I used C# in Unity to code the back and forth movement of the enemies.

Before this project, I had no experience with c#. However, Game Creation Society had different tutorials to help ease us into using Unity. As a team, we were also able to learn from each other and received help from our program leader. Through this project, I was able to experiment in a new language and also learn how to design games in Unity.
