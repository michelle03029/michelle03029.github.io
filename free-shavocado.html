Free Shavocado

Running Game

The Conception of "Free Shavocado!""

Designed and Programmed by: Michelle Chen

April - May, 2018

Storyboard

Going into this project, I wanted to tackle game development with PyGame using Object Oriented Programming. One of my favorite phone games is Ovenbreak which is gingerbread themed so I wanted to create a similar scene but with one of my favorite foods: guacomole.

"Free Shavocado!"" is an endless running game where the player (an avocado) can jump or duck to escape obstacles that can turn the avocado into guacomole.
